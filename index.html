Welcome to Pet bit melody

This is the home page for the pet bit melody library.

The pet bit melody library is a python library that aims to let you compose your own tracks so that your character on screen can sing them.


Don't have Desktop pets?

No problem, you can download it here: desktop-pets.com


Already have it, and it is running, then you can use this (to come) web interface to try it out.
